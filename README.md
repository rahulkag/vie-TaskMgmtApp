## Project Setup
1. Clone
```sh
git clone https://github.com/rahulkag/vue-TaskMgmtApp.git

npm install
```

### Compile and Hot-Reload for Development

Start local server run
```sh
npm run dev
```
To start json server
```sh
npx json-server --watch db.json
```
### Compile and Minify for Production

```sh
npm run build
```
